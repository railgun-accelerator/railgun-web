协议列表

以下是 Railgun 支持的各种连接方式, 如果你对网络相关有足够的了解, 并且希望在 Railgun 没有直接写说明文档的其他平台上运行, 请参照这个列表进行配置.

游戏主机通常支持 HTTP 代理, 路由器通常支持 PPTP 或 L2TP VPN. ChromeOS 对 HTTP 代理和 VPN 都支持. Chromecast 没有任何代理功能, 需要在上级网关上进行设置.

如果发现有某个设备提供了代理/VPN功能, 但是 无法正常连接 Railgun, 请联系 support@railgun.ac
