IPSec HyBrid RSA

  • 打开 设置 - 更多 - VPN
  • 点击 添加 VPN 配置文件 (+)
  • 输入
属性 值
名称 Railgun
类型 IPSec Hybrid RSA
服务器地址 {{user.zone}}.lv5.ac
显示高级选项 √
转发路线 
* 请在 Android 设备上用浏览器访问这个页面, 长按, 全选, 复制, 切回设置页面粘贴
    * 如果你不需要国内地址分流功能, 则不需要填写转发路线
  • 连接 VPN
属性 值
用户名 {{user.username}}
密码 {{user.sub_password}}

  • 点击 添加 VPN 配置文件 (+)
  • 输入
属性 值

* 始终开启的 VPN 不支持国内地址分流功能
  • 连接
属性 值

密码 {{user.sub_password}}

AnyConnect (OpenConnect)

  • 从 https://play.google.com/store/apps/details?id=app.openconnect 安装 OpenConnect 客户端并运行

  • 点击 添加配置文件 (+)

  • 输入

属性 值
服务器地址 {{user.zone}}.lv5.ac
批处理模式 Prompt for empty fields only
拆分隧道模式 On - DNS uses VPN
拆分隧道网络 
* 请在 Android 设备上用浏览器访问这个页面, 长按, 全选, 复制, 切回 OpenConnect 设置页面粘贴
  • 按 back 键, 连接
属性 值
Username {{user.username}}
password {{user.sub_password}}

shadowsocks

注意: 从 2015-11-27 我们才增加了 shadowsocks 支持, 这之前的用户没有开启 shadowsocks, 请点击 "重置服务密码" 以开启 shadowsocks 服务

  • 从 https://play.google.com/store/apps/details?id=com.github.shadowsocks 安装 Shadowsocks 客户端并运行

  • 点击 添加配置文件

  • 扫描二维码

  • 或者手动设置
